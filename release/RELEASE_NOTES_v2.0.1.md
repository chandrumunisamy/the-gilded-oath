# The Gilded Oath v2.0.1

The first public Android build of The Gilded Oath.

## Highlights

- Touch-first portrait card roguelite
- Light/Dark card-state decisions with limited Flips and Swaps
- Three-act seeded routes with encounters, events, shops, rewards, and bosses
- Thirty Powers and Seals
- Momentum, Oath XP, ranks, Daily Oaths, and cosmetic table themes
- Context-routed music and tactile sound effects
- Offline play; the Android export requests no Internet permission
